# MelissArte Photos

## Objetivo

Crear una experiencia para que los invitados compartan y disfruten las fotos de un evento mediante un código QR.

## Prioridad

- Sencillo

### Experiencia pública
- Elegante
- Emocional
- Fácil de usar
- Optimizada para celular
